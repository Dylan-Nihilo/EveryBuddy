# EveryBuddy Showcase

Static gallery page for reviewing two EveryBuddy product states:

1. Collectible pet cards
2. tmux sidecar runtime snapshots

## Open Locally

Direct open:

```bash
open /Users/dylanthomas/Desktop/projects/EveryBuddy/showcase/index.html
```

or run a local static server:

```bash
cd /Users/dylanthomas/Desktop/projects/EveryBuddy/showcase
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173
```
